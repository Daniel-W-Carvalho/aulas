Oque é API?

Uma API (Interface de Programação de Aplicações, ou Application Programming Interface) é um conjunto de regras e ferramentas que permite que diferentes sistemas, aplicativos ou partes de um software se comuniquem entre si. No contexto do JavaScript, as APIs desempenham um papel fundamental, pois oferecem meios para interagir com o navegador, acessar serviços externos, ou trabalhar com funcionalidades específicas.
